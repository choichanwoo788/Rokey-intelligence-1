# Intelligence 1

AMR leader/follower tracking and YOLO-based perception code for the Intelligence 1 project.

## Contents

- `src/src/`: Python nodes and scripts for AMR navigation, YOLO detection, tracking, and classifier experiments.

## Archived Materials

Videos, presentation files, PDFs, images, compressed files, model binaries, and original downloaded bundles were moved to `../아카이브`.
